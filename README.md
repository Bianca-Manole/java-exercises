# Java Exercises

## Descriere
Acest repository conține exercițiile mele practice în Java, realizate pe măsură ce învăț conceptele de bază și intermediare.

Scopul lui este să urmăresc progresul meu, să exersez logica de programare și să păstrez exemple clare pentru diferite tipuri de probleme.

## Ce conține
Repository-ul include exerciții și exemple pentru:

- variabile și tipuri de date
- condiții (`if`, `else`, `switch`)
- bucle (`for`, `while`)
- metode
- array-uri și liste
- map-uri
- clase și obiecte
- constructori
- moștenire și concepte OOP de bază
- exerciții de logică
- probleme practice și mici modele

## Scop
Prin acest repository:
- exersez constant Java
- îmi organizez exercițiile într-un singur loc
- urmăresc evoluția mea de la concepte simple la probleme mai complexe

## Observație
Acest proiect este folosit pentru învățare și practică, deci unele fișiere sunt exerciții independente, nu părți ale unei singure aplicații.
